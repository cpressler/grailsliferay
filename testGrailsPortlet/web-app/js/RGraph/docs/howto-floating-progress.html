HOWTO: Create a floating progress bar

RGraph doesn't support floating progress bars as an option, though by using the CSS3 color rgba() color selector it can be achieved. It works by setting the inner strokestyle and first color to transparent so that they cannot be seen. If using tooltips then the first tooltip will also need to be set to null.

1. The original progress bar

Here the original progress bar is shown before altering the colors.

[No canvas support] 
<script>
    var hprogress = new RGraph.HProgress('cvs1', [5,4,2,4], 20);
    hprogress.Set('chart.margin', 5);
    hprogress.Set('chart.tickmarks', 10);
    hprogress.Set('chart.tickmarks.inner', true);
    hprogress.Set('chart.colors', ['red','#0f0','yellow','pink']);
    hprogress.Set('chart.tooltips', ['Gary','Joel','Peter','Rich']);
    hprogress.Draw();
</script>

2. The floating progress bar

Now the colors and values are set accordingly to make the progress appear as though it's floating. You will need to calculate the first value so that the first section of the bar is positioned correctly. The first color is set to transparent (rgba(0,0,0,0), the chart.strokestyle value is set to transparent and the first tooltip (if tooltips are used) is set to null.

[No canvas support]
[No canvas support] 
<script>
    var hprogress = new RGraph.HProgress('cvs2', [5,4,2,4], 20);
    hprogress.Set('chart.margin', 5);
    hprogress.Set('chart.tickmarks', 10);
    hprogress.Set('chart.tickmarks.inner', true);
    hprogress.Set('chart.colors', ['rgba(0,0,0,0)','#0f0','yellow','pink']);
    hprogress.Set('chart.tooltips', [null,'Joel','Peter','Rich']);
    hprogress.Set('chart.strokestyle.inner', 'rgba(0,0,0,0)');
    hprogress.Draw();

    var vprogress = new RGraph.VProgress('cvs3', [5,4,2,4], 20);
    vprogress.Set('chart.margin', 5);
    vprogress.Set('chart.tickmarks', 10);
    vprogress.Set('chart.tickmarks.inner', true);
    vprogress.Set('chart.colors', ['rgba(0,0,0,0)','#0f0','yellow','pink']);
    vprogress.Set('chart.tooltips', [null,'Joel','Peter','Rich']);
    vprogress.Set('chart.strokestyle.inner', 'rgba(0,0,0,0)');
    vprogress.Draw();
</script>
