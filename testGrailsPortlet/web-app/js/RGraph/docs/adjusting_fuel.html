Adjusting your charts interactively - Fuel chart

[No canvas support]

The Fuel chart has the facility to be adjusted. There are three custom RGraph events associated with it:

  • onadjustbegin
  • onadjust
  • onadjustend

See below for an example of using these events. 


<script>
    window.onload = function (e)
    {
        var fuel = new RGraph.Fuel('cvs', 0,100,78);
        fuel.Set('chart.adjustable', true);
        fuel.Draw();
    }
</script>

The RGraph adjusting events

There are three RGraph events associated with adjusting. The onadjustbegin event fires when adjusting begins. Much like the onmousedown event. The onadjust event fires during adjusting, much like the onmousemove event. The onadjustend event fires when adjusting is finished, much like the onmouseup event.

Some examples of using the three events. Each gets the shape information (ie the applicable bar) from the RGraph registry. 

RGraph.AddCustomEventListener(fuel, 'onadjustbegin', function (obj) {cl(obj.value);});
RGraph.AddCustomEventListener(fuel, 'onadjust', function (obj) {cl(obj.value);});
RGraph.AddCustomEventListener(fuel, 'onadjustend', function (obj) {cl(obj.value)});
